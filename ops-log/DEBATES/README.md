# DEBATES — 週次改善の議事録

このディレクトリは **週次改善 Routine** が行うブレスト・対立議論・ジャッジ裁定の逐語議事録を保存する場所。
ファイル名: `YYYY-MM-DD.md`（改善を実行した日付）。

## ファイル構成 (各議事録のフォーマット)

```markdown
# 週次改善議事録 YYYY-MM-DD

## 読んだ日次ログ

- YYYY-MM-DD (X 件)
- ...

---

## フェーズA: ブレスト議事録

### ラウンド 1

**🔭 ソースハンター T1**:  
...

**🙋 ユーザー代弁 T2**:  
...

**💡 斬新枠 T3**:  
...

### ラウンド 2

...

---

## フェーズB: 対立議論

### 論点 1: <論点タイトル>

**🔵 推進派 T1**:  
...

**🔴 懐疑派 T2**:  
...

...

**⚖️ ジャッジ裁定**:  
結論: **やる / やらない / 保留**  
...

---

## 実施した改善

- [実施した改善の概要]
```

## 制約 (IMPROVE_PROMPT.md 準拠)

- **ブレスト最大3ラウンド**（1ラウンド = 3レンズ×1発言）
- **対立議論は1論点あたり最大4ラリー**（1ラリー = 推進派↔懐疑派の往復1回）
- **論点は最大3件**
- 議論は要約せず逐語で記録する
- ジャッジは1論点につき1回のみ
